__Features__
- Dynamic Island Generation
- Complete Global/Island/Sessional Leaderboard Support
- PlaceholderAPI Support
- 99% Customizable Messages
- Block GUI   
...and many more!

__Dependencies__
- WorldEdit (Required)
- PlaceholderAPI (Optional)
